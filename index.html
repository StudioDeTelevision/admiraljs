About

Admiral JS is a admin web app entirely written in javascript with dependencies on bootstrap, jquery, backbone, underscore, on a module logic with REQUIREJS

The data connection is ensured by a RestFul Api + File Upload Server based on NodeJS (but it might easily be ported to other systems)

Installation

Requirement: a Nodejs installation , a mongodb host.

1. npm install admiraljs

2. var admiral=require('admiraljs');

3. admiral.config({....})
see all configuration options

4. admiral.start();
